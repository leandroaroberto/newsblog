## News Blog Style
By Leandro A. Roberto

This project is a blog style site using laravel 5.4 implementing the functionalities below:

- Auto-Summary
- RSS Feed
- PDF Generation
- Contact form Markdown Mailable
- Picture upload
- Create user with e-mail validation 
- Dusk unity tests
